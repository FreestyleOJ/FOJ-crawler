## Description
输入两个整数A和B，输出他们的和<br>
              
            
## Input Description
输入为一行，包含两个整数A，B。数据保证A与B都在2^31-1的范围内<br>

            
## Output Description
输入A与B的和，数据保证A与B的和在2^31-1的范围内<br>
              
            
## Sample Input
1 3<br>

            
## Sample Output
4<br>
